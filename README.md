## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Krishna-Lohith/Python.git
cd Python
```

### 2. Install Dependencies

```bash
pip install notebook pandas plotly cufflinks
```

### 3. Launch Jupyter

```bash
jupyter notebook
```

### 4. Explore

Navigate to any subdirectory and open the `.ipynb` files to view and run the code cells interactively.
